# Contribution

Mau bantu ngembangin Tabungin? Mantap.

## Cara kontribusi

1. Fork atau buka project ini di Replit
2. Bikin perubahan seperlunya
3. Cek hasil di preview
4. Pastikan build aman
5. Kirim perubahan

## Saran kontribusi

- rapihin UI
- tambah fitur statistik
- bikin validasi input lebih bagus
- tambah tema baru
- tingkatkan keamanan dan pengalaman pengguna

## Aturan umum

- jangan merusak alur enkripsi
- jangan simpan data sensitif mentah
- jaga konsistensi desain
