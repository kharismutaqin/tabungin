# Development

Proyek ini pakai:

- Vite
- React + TypeScript
- Tailwind CSS
- Supabase

## Jalankan lokal

```bash
pnpm install
pnpm --filter @workspace/tabungin dev
```

## Build

```bash
pnpm --filter @workspace/tabungin build
```

## Catatan

- Dev server jalan di `0.0.0.0`
- Preview Replit butuh host yang mengizinkan proxy
- Data sensitif dienkripsi di sisi client
